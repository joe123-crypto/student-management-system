# Setup and Local Development

## Prerequisites
- Node.js (LTS recommended)
- npm

## Install
```bash
npm install
```

## Environment
Copy `.env.example` into `.env` and configure:
- `NEXT_PUBLIC_DEMO_MODE`
- `NEXT_PUBLIC_ATTACHE_PASSWORD`

## Run
```bash
npm run dev
```

## Useful Commands
- `npm run lint`
- `npm run docs:check`
